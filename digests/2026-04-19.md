## Deep Dive of the Day

**Harrison Chase's three-layer model of agent learning (Model / Harness / Context)** is the single most interview-valuable concept in today's digest. 
The three main layers are: the Model (weights themselves), the Harness (code that drives the agent plus any instructions or tools always part of it), and Context (additional instructions and skills that live outside the harness and can configure it)
. This framework matters because most FAANG agent interviews test whether you can reason about *where* a behavioral improvement should live — do you fine-tune, change the scaffold, or inject a skill? — and the wrong answer usually reveals fuzzy thinking. Worth digging deeper into: (1) 
papers on harness optimization like "Meta-Harness: End-to-End Optimization of Model Harnesses" where the agent runs in a loop, gets evaluated, and a coding agent suggests changes to the harness code
; (2) tenant-level vs. agent-level context learning and how it intersects with memory architectures like Mem0g's graph memory; and (3) how Anthropic's Agent Skills and OpenAI's newly added "reusable agent skills" in the Responses API are converging on context as the primary learning surface — because it's cheap, portable, and observable compared to retraining weights.
